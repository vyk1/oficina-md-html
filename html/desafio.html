Desafio: Criar uma página de perfil pessoal

Objetivo: Desenvolver uma página em HTML que apresente informações sobre você, incluindo seu nome, foto, uma breve descrição, suas habilidades e links para suas redes sociais.

Requisitos:

  1. Crie uma estrutura básica em HTML, incluindo as tags <html>, <head> e <body>.
  2. Adicione um título à página dentro da tag <head>.
  3. Dentro da tag <body>, crie uma seção para o cabeçalho da página, onde você irá exibir seu nome e uma foto sua.
  4. Abaixo do cabeçalho, crie uma seção para a descrição, onde você irá escrever uma breve introdução sobre si mesmo.
  5. Em seguida, crie uma seção para listar suas habilidades. Utilize listas HTML (<ul> e <li>) para organizar as habilidades.
  6. Por fim, adicione ícones/link para suas redes sociais. Você pode utilizar ícones de bibliotecas como Font Awesome (https://fontawesome.com/) ou Ionicons (https://ionicons.com/), e adicionar os links para suas redes sociais.

Dicas:

  • Utilize CSS para estilizar a página e torná-la visualmente atraente.
  • Pesquise sobre as tags HTML relevantes para cada seção (por exemplo, <h1>, <img>, <p>, <ul>, <li>, etc.).
  • Utilize atributos HTML, como src, alt, href, para especificar o caminho da imagem, texto alternativo para a imagem e URLs dos links, respectivamente.

Lembre-se de que este é apenas um desafio proposto para desenvolver suas habilidades em HTML. Sinta-se à vontade para adicionar mais elementos e personalizar a página conforme desejar. Divirta-se criando sua página de perfil pessoal!
